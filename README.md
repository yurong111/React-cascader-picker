# React-cascader-picker

主要是封装了[m-cascader](https://github.com/react-component/m-cascader)组件。一个级联的地区选择器组件。

使用：
```
  yarn install
  yarn start
```
props属性有：
```
  data            //数据结构
  visible         //是否显示
  value           //值
  onChange        //点击确定事件
  onScrollChange  //滚动选择事件
  onDismiss       //取消选择事件
  togglePicker    //切换显示事件
```
